# solo-provisioner

A Go-based tool for provisioning Hedera network components in a streamlined and automated way.

## Features

- Automated resource provisioning
- Simple configuration
- Extensible and modular design

## Installation

```sh
go install github.com/yourusername/solo-provisioner@latest
```

Or clone the repository:

```sh
git clone https://github.com/yourusername/solo-provisioner.git
cd solo-provisioner
go build
```

## Usage

```sh
task run -- [flags]
```

Example:

```sh
task run -- --config=config.yaml
```
